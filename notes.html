HTML Tags Notes

Home Notes
<h1>
Defines the main heading for the entire page.
<p>
Defines a paragraph of text.
<ul>
Defines a list of items where order does not matter, typically displayed with bullets.
<li>
Defines a list item in an ordered or unordered list.
<a>
Defines a hyperlink, used to link to another page or resource.
